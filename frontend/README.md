# React (TypeScript) frontend

## Important commands (make sure they're ran inside `/frontend` or its subfolders!)

### To be ran upon cloning repo

```bash
npm install
```

### To preview the frontend in developer mode with Vite

```bash
npm run dev
```

### To build locally with TypeScript compiler and Vite

```bash
npm run build
```

### To preview local build with Vite **(Run above command first!)**

```bash
npm run preview
```

### To manually audit code in current folder with ESLint

```bash
npm run lint
```

### To manually reformat code in current folder with Prettier

```bash
npm run format
```
